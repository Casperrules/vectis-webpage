Simple to Use

# 1. Start the Server
./vectis serve

# 2. Upload Data (CSV or TXT)
./vectis upload --file products.csv --token "YOUR_TOKEN"

# 3. Search
./vectis search --query "fast gaming laptop" --token "YOUR_TOKEN"
